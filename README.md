# Codename

## Installation

Installing Tauri dependencies :

```bash
sudo apt update
sudo apt install libwebkit2gtk-4.1-dev \
  build-essential \
  curl \
  wget \
  file \
  libxdo-dev \
  libssl-dev \
  libayatana-appindicator3-dev \
  librsvg2-dev
```

then

```bash
npm install
```

## Start web

```bash
npm run dev
```

## Start desktop

```bash
npm run tauri dev
```
